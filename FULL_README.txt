LEAD GENERATION & AUDIT MACHINE - FULL (Windows)

REQUISITI (Windows):
- Node.js 18+ (LTS consigliato)
- Python 3.11+ (consigliato 3.11/3.12)

AVVIO (1 click):
1) Estrai lo ZIP in una cartella (es. Desktop)
2) Doppio click su: START_SOFTWARE.bat
3) Attendi: alla prima esecuzione puo' richiedere alcuni minuti (install dipendenze)
4) Apri il browser su: http://localhost:3000

NOTE:
- Se un'altra istanza e' gia' avviata su 3000/8000, chiudila e rilancia.
- Se il browser non si apre, usa comunque: http://localhost:3000

LOG (in caso di problemi):
- start_software.log
