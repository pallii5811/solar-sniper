LEAD GENERATION & AUDIT MACHINE - DEMO (MILANO)

Questa DEMO e' bloccata su:
- Citta': Milano
- Categorie consentite: Ristoranti, Commercialisti

REQUISITI (Windows):
- Node.js (LTS consigliato)
- Python 3.x (consigliato 3.11/3.12)

AVVIO (1 click):
1) Apri la cartella della DEMO
2) Doppio click su: START_DEMO_CLIENTE.vbs
3) Attendi l'apertura del browser su: http://localhost:3001

NOTE:
- La prima esecuzione puo' richiedere alcuni minuti (install dipendenze).
- Se un'altra istanza del software e' gia' avviata su 3001/8001, chiudila e rilancia la DEMO.

LOG (in caso di problemi):
- demo_start.log
- demo_frontend_out.log / demo_frontend_err.log
- demo_backend_out.log / demo_backend_err.log
